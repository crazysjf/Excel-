# 安装
算账宏.xla 必须位于 D:\报表处理宏\，否则算账公式引用会出问题。
目前没有好办法解决，据说excel 2014版已经可以解决。

安装：excel按钮 => excel选项 => 加载项 => 转到 => 浏览，选xla文件打钩即可。
卸载：上面位置，去掉对应的钩即可。


# 注意事项：
add-in被加载后，加载项的菜单好像并不能显示出来，需要手动执行创建菜单的宏：
在 开发者工具 => 宏，宏名输入：Workbook_AddinInstall，执行。

同样，add-in被卸载后菜单仍然存在。手动执行宏：Workbook_AddinUninstall

# 开发
从0开始的开发过程：创建xlsm文件，编辑里面的vba代码，保存为xla文件。之后可以直接对xla文件进行编辑，xlsm文件可以丢弃。

无法进行完整的版本控制。
直接打开了任何一个文件的excel里面，去VB里面编辑即可。保存后add-in就被更新。
